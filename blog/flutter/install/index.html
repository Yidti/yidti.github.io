Flutter - macOS install

By Sola 11 March 2023

跨平台開發的首選

目前開發跨平台的首選應該是Flutter吧？
讓我們來試看看如何來安裝跨平台開發工具Flutter吧

安裝Flutter

其他平台安裝方式請參考官方網站 https://docs.flutter.dev/get-started/install

  1. 滿足系統需求之外，由於筆者電腦是M1，所以需要額外執行以下
sudo softwareupdate --install-rosetta --agree-to-license
  1. 下載Flutter

  1. 解壓縮檔案並移動到此資料夾放置希望存放的地方

  2. 新增flutter到你的路徑(Path) - 暫時加入

export PATH="$PATH:`pwd`/flutter/bin"

確認路徑 which flutter

PS. 永久更新路徑: https://docs.flutter.dev/get-started/install/macos#update-your-path

  1. 執行flutter doctor檢查是否還缺任何的dependencies
flutter doctor


例如我遇到的是dart的路徑有誤，去找一下是否有人也遇到跟我一樣問題
https://ithelp.ithome.com.tw/articles/10237569 解決方法是執行 brew uninstall dart
解除安裝之前單獨下載dart的版本

  1. 若你遇到不同的問題，請逐一檢查排除直到以下畫面出現

編輯器的選擇，VSCode or Android Studio or IntelliJ

可以參考官方設定Editor的插件設定
https://docs.flutter.dev/get-started/editor
